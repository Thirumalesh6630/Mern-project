## üõ†Ô∏è Technical Requirements

### Mandatory Features (All Projects Must Include):

1. **Backend (Node.js + Express.js)**
    - RESTful API with at least 5 different endpoints
    - Proper routing and middleware implementation
    - Error handling middleware
    - Input validation and sanitization
    - Environment variables configuration
2. **Database (MongoDB) or SQL**
    - At least 2 related collections/models
    - Proper schema design with validation
    - CRUD operations implementation
    - Data relationships (references or embedded documents)
3. **Frontend (React.js)**
    - At least 5-7 components with proper component hierarchy
    - React Hooks (useState, useEffect, useContext, custom hooks)
    - React Router for navigation (minimum 3 routes)
    - Form handling with validation
    - Responsive design (mobile-friendly)
    - Loading states and error handling
    - Clean and intuitive UI/UX
4. **Additional Requirements**
    - User authentication (JWT or session-based)
    - Protected routes
    - Toast notifications or alerts for user feedback
    - Code organization and clean architecture
    - README.md with setup instructions
